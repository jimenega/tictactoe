# tictactoe
This program is a game of Tic-Tac-Toe written in java.<br />
X player goes first.<br />
The coordinates are:<br />
  bottom/left (1 1)<br />
  top/right (3 3)<br />
  
Use the following coordinates to play:<br /> 
(1 3) (2 3) (3 3)<br /> 
(1 2) (2 2) (3 2)<br /> 
(1 1) (2 1) (3 1)<br />

Example step:<br />
Enter the coordinates: 1 3
